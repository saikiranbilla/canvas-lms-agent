# Chat completion

Copy page

Complete guide to integrating Keywords AI with OpenAI SDK for seamless LLM gateway usage.

Copy page

This integration is for the **Keywords AI gateway**.

## [​](https://docs.keywordsai.co/integration/development-frameworks/llm_framework/openai/openai-sdk\#overview)  Overview

OpenAI SDK provides the most robust integration method for accessing multiple model providers.Since most AI providers prioritize OpenAI SDK compatibility, you can seamlessly call all 250+ models available through the Keywords AI platform gateway.

## [​](https://docs.keywordsai.co/integration/development-frameworks/llm_framework/openai/openai-sdk\#quickstart)  Quickstart

### [​](https://docs.keywordsai.co/integration/development-frameworks/llm_framework/openai/openai-sdk\#step-1:-install-openai-sdk)  Step 1: Install OpenAI SDK

- Get a Keywords AI API key
- Add your provider credentials
- Install packages

Python

TypeScript

Go

Copy

```
pip install openai
```

### [​](https://docs.keywordsai.co/integration/development-frameworks/llm_framework/openai/openai-sdk\#step-2:-initialize-client)  Step 2: Initialize Client

Python

TypeScript

Go

Copy

```
from openai import OpenAI

client = OpenAI(
    base_url="https://api.keywordsai.co/api/",
    api_key="YOUR_KEYWORDS_AI_API_KEY",  # Get from Keywords AI dashboard
)
```

### [​](https://docs.keywordsai.co/integration/development-frameworks/llm_framework/openai/openai-sdk\#step-3:-make-your-first-request)  Step 3: Make Your First Request

Python

TypeScript

Go

Copy

```
response = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "Hello, world!"}],
)
print(response.choices[0].message.content)
```

### [​](https://docs.keywordsai.co/integration/development-frameworks/llm_framework/openai/openai-sdk\#step-4:-see-your-log-on-platform)  Step 4: See your log on [platform](https://platform.keywordsai.co/platform/requests?sort_by=-timestamp)

## [​](https://docs.keywordsai.co/integration/development-frameworks/llm_framework/openai/openai-sdk\#switch-models)  Switch models

Python

Typescript

Copy

```
# OpenAI GPT models
model = "gpt-4o"
# model = "claude-3-5-sonnet-20241022"
# model = "gemini-1.5-pro"

response = client.chat.completions.create(
    model=model,
    messages=[{"role": "user", "content": "Your message"}],
)
```

See the [full model list](https://platform.keywordsai.co/platform/models) for all available models.

## [​](https://docs.keywordsai.co/integration/development-frameworks/llm_framework/openai/openai-sdk\#supported-parameters)  Supported parameters

### [​](https://docs.keywordsai.co/integration/development-frameworks/llm_framework/openai/openai-sdk\#openai-parameters)  OpenAI parameters

We support all the [OpenAI parameters](https://docs.keywordsai.co/api-endpoints/develop/gateway/chat-completions#openai-compatible-parameters). You can pass them directly in the request body.

Python

Typescript

Copy

```
response = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "Tell me a story"}],
    temperature=0.7,          # Control randomness
    max_tokens=1000,          # Limit response length
    top_p=0.9,               # Nucleus sampling
    frequency_penalty=0.1,    # Reduce repetition
    presence_penalty=0.1,     # Encourage topic diversity
    stream=True,             # Enable streaming
)
```

### [​](https://docs.keywordsai.co/integration/development-frameworks/llm_framework/openai/openai-sdk\#keywords-ai-parameters)  Keywords AI Parameters

[Keywords AI parameters](https://docs.keywordsai.co/api-endpoints/develop/gateway/chat-completions#keywords-ai-parameters) can be passed for better handling and customization.

Python

Typescript

Copy

```
response = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "Tell me a story"}],
    extra_body={
        "customer_identifier": "user_123",           # Track specific users
        "fallback_models": ["gpt-3.5-turbo"],       # Automatic fallbacks
        "metadata": {"session_id": "abc123"},        # Custom metadata
        "thread_identifier": "conversation_456",     # Group related messages
        "group_identifier": "team_alpha",           # Organize by groups
    }
)
```

## [​](https://docs.keywordsai.co/integration/development-frameworks/llm_framework/openai/openai-sdk\#azure-openai)  Azure OpenAI

To call Azure OpenAI models, instead of using azure OpenAI’s client, the easier way is to use the OpenAI client.

Setup

Python

Copy

```
1. Go to [Keywords AI Providers](https://platform.keywordsai.co/platform/api/providers)
2. Add your Azure OpenAI credentials
3. Configure your Azure deployment settings
4. Use Azure models through the same Keywords AI endpoint
```
